## 欢迎来到我的技术博客！

很高兴在这里遇见你 👋 这是一个用 Next.js 构建的静态技术博客。

### 为什么写博客？

写博客有几个好处：

- **加深理解**：费曼学习法，教是最好的学
- **知识沉淀**：把经验记录下来，避免重复踩坑
- **交流分享**：和其他开发者一起进步

### 这里会写什么？

- Next.js / React 技术深度解析
- 前端工程化实践
- TypeScript 高级用法
- Tailwind CSS 技巧
- 以及一些踩坑总结
